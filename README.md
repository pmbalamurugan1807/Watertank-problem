# Watertank-problem

Entire project is done in 1 html file ( watertank.html )

Water Tank Problem
Given “n” always greater than -1, representing the block height. Now compute
the units of water stored in-between the blocks. Build a Web Application
(Frontend Solution) using Vanilla JavaScript and HTML/CSS to represent the
solution. 


Open the html file and you can input the blocks info as shown below and get how
many blocks of water is stored out of given blocks input
